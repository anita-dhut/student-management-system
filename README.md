# Student Management System

A Java console-based application developed to manage student records efficiently.

## Features

* Add new student details
* View all students
* Search student by ID
* Delete student records

## Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)

## How to Run

1. Open the project in Eclipse.
2. Run `Main.java`.
3. Use the console menu to manage students.
